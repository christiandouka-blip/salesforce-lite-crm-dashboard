# Salesforce CRM Clone PRD

## Objective

Build a high-fidelity Salesforce CRM Clone using HTML, CSS, and JavaScript.

The goal is to recreate the Salesforce login experience and CRM dashboard experience as closely as possible using the provided reference screenshots and PDFs.

This project must feel like a real clickable product prototype.

## Branding Rule

The clone must show the Salesforce-style cloud logo with the word "Clone" next to it so users know this is a school project clone, not the real Salesforce product.

## Login Screens to Clone

- Main Login Page
- Forgot Password Page
- Verify Identity Help Page
- Use Email Address Recovery Page
- Free CRM Signup Page
- Login Dropdown Menu
- Marketing Cloud Login Page
- Trailblazer Login Page
- Custom Domain Login Page

## Dashboard Screens to Clone

- Home Dashboard
- Accounts Page
- Leads Page
- Cases Page
- Contacts Page

## Core Requirement

Every button, link, form, dropdown, sidebar item, and navigation item must be clickable and connected.

## Success Criteria

A user should be able to start at the login page, move through the login-related screens, log in successfully, and land on the Salesforce-style dashboard.
